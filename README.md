# GLB Studio

An Android application developed for managing and viewing 3D GLB models offline.

## Features
* **3D Rendering**: View GLB models using SceneView.
* **Offline Storage**: Local persistence using Room Database.
* **Role-Based Access**: Specialized views for Admin and User roles.
* **Modern UI**: Built entirely with Jetpack Compose.

## Tech Stack
* **Language**: Kotlin 2.1.0
* **Database**: Room (KSP)
* **3D Engine**: SceneView
* **UI**: Jetpack Compose
* **Dependency Management**: Version Catalogs (libs.versions.toml)

## Hardware Tested
* Samsung S22 Ultra (Android 14/15)
